Guiding Your Path to Success: Graphic Designer's Roadmap

Week 1

Introduction and Installation of Software

  • Explanation about system configuration
  • Photoshop requirements for Windows/MacOS
  • Downloading and installation of Photoshop versions
  • Actions & Plugins
Week 2

Workspace & Shortcuts

  • Software layout
  • Panels customizing and managing panels, docking, and arranging for optimal workflow.
  • Basic and advanced keys
  • Shortcuts mastering shortcuts for common tasks to speed up the design process.
Week 3

Different Formats in Photoshop & Tool Box

  • Opening Different Formats
  • Tool Box Overview
  • Tool Presets setting up and saving tool presets for efficient workflow.
  • Advanced Tool Techniques
Week 4

Menu, Channels, Layers

  • File, Edit, Image, and Layer menus
  • Channels and layers management
  • Layer styles and effects
  • Advanced Layer Techniques
Week 5

Layer Masks, Text, Sizes, and Resolutions

  • Vector and clip masks
  • Text effects and styles
  • Poster and printing sizes
  • Color correction and grading
Week 6

Image Quality, Blur Tools, Logo and Portfolio Design

  • Image quality enhancement
  • Blur tools and effects
  • Logo design basics
  • Creating a design portfolio
Week 7

Adobe Illustrator Basics

  • Installation and configuration
  • Basic shapes and tools
  • Combining shapes and objects
  • Typography and text effects
Week 8

Adobe InDesign Fundamentals

  • Installation and configuration
  • Document setup and navigation
  • Text formatting and special characters
  • Exporting for print and digital content
